Calculadora desenvolvida para fins acadêmicos, utilizando HTML, CSS e Javascript.

Funções elaboradas:
1-Contas envolvendo os operadores básicos (+ , - , * , /).
2-Inserção dos números clicados na tela em tempo real.
3-Limpeza de campo ao clicar na tecla C para resetar.
4-Gerador de histórico que guarda todos os resultados calculados.
5-Contas envolvendo porcentagem (utilizado REGEX para capturar os campos).
6-Opção de inverter o sinal do número atual inserido.
7-Opção de deletar apenas um número ou sinal ao invés de apagar o campo inteiro.
8-Utilização viabilizada via cliques na tela e através de uso do teclado.
9-Rodapé com contato para minhas redes profissionais.
